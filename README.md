# Stories LB AI Assistant ☕🤖📍

An AI-powered website assistant built for Stories LB coffee shop that helps customers get instant answers, discover products, and locate the nearest branch using their current location.

---

## 🚀 Project Overview

The Stories LB AI Assistant combines conversational AI with location-based services to create a smarter customer experience.

Customers can:
- Chat with an AI assistant directly from the website
- Ask questions about the coffee shop
- Get information about products and services
- Find the nearest Stories LB branch based on their current location
- Receive instant support 24/7
